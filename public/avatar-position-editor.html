🎨 아바타 위치 편집기

1. 베이스 캐릭터 선택 → 슬롯 탭 선택 → 아이템 선택
2. 미리보기에서 드래그로 위치, 오른쪽 아래 핸들로 크기 조정
3. 또는 우측 슬라이더/입력으로 정밀 조정
4. 각 아이템 fine-tune 후 JSON 다운로드해서 Claude에게 전달

미리보기

• 보라색 점선 박스 = 선택된 아이템
• 박스 안 드래그 = 위치 이동
• 오른쪽 아래 점 = 크기 조정
• 다른 슬롯도 동시 합성되어 보임 (헤어+모자+안경+의상 등 같이 확인)
